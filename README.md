# AutoSchedule

Student weekly planner demo (React + TypeScript + Vite). Setup wake/sleep, activities, and fixed events, then generate an editable weekly calendar. State is stored in `localStorage`.

## Run

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

## Notes

- **Demo login** — any username/password works; not suitable for production auth.
- **Routes** — `#/`, `#/setup`, `#/schedule` (refresh restores the current page).
- **QA report** — see [STUDENT_REPORT.md](./STUDENT_REPORT.md) for implementation changes and the testing strategy.
